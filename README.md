# 🎯 Number Guessing Game (Python)

A simple command-line number guessing game built using Python.  
The program randomly selects a number between 1 and 100, and the player has to guess it.

---

## 🚀 Features

- Random number generation between 1 to 100
- User input-based guessing
- Hints provided:
  - "Higher number please!!"
  - "Lower number please!!"
- Tracks number of attempts

---

## 🛠️ Technologies Used

- Python
- Built-in `random` module
